### [ğŸš¨ Agentic RAG for ITSM Automation](https://github.com/Vinaykumaryadi/itsm-rag-ultimate)
*An intelligent support agent that reasons, retrieves, and self-corrects.*



* **The Problem:** Traditional RAG fails on vague user queries or fragmented enterprise logs.
* **The Solution:** Built a cyclic **Agentic reasoning loop** that mimics a human technician's troubleshooting process.
* **The Tech:** LangGraph, Llama 3.3 (via Groq), FAISS, HuggingFace Embeddings.
* **Key Outcome:** Achieved sub-second reasoning and significantly reduced hallucinations by grounding responses in verified Incident IDs.
